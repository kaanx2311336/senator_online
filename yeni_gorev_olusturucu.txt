🏛️ ROMA CITY BUILDER - YENİ GÖREV OLUŞTURUCU - TAM SİSTEM REFERANS DOKÜMANI
================================================================================
Bu dosya, Roma City Builder otomasyon sisteminin TÜM iç mekanizmalarını açıklar.
Herhangi bir AI'ya bu dosyayı referans gösterip yeni dalga/faz eklemesini
isteyebilirsiniz.

PROJE: Roma İmparatorluğu Şehir Kurma (3D City Builder)
TEKNOLOJİ: Three.js + Tailwind CSS + GSAP + Vanilla JS (tek sayfa, sunucusuz)
FRAMEWORK: React/Vue/Angular YASAK. Three.js + Tailwind CDN üzerinden.

GÖRSEL TEMA:
  - Ferah, aydınlık gündüz vakti, canlı yeşil çimen zemin
  - Bembeyaz mermer duvarlı, kiremit (kırmızı) çatılı Roma mimarisi
  - Gölgeler KAPALI, renkler temiz, Low-poly/Retro strateji tarzı
  - renderer.shadowMap.enabled = false
  - renderer.setPixelRatio(1)
  - renderer.setClearColor() ile gökyüzü/çimen rengi sabit

KRİTİK TEKNİK KURALLAR (Siyah Ekran Engelleme):
  1. renderer.shadowMap.enabled = false (gölge tamamen kapalı)
  2. renderer.setPixelRatio(1) (piksel oranı sabit)
  3. renderer.setClearColor(0x87CEEB) ile arka plan rengi sabit
  4. window.onload KULLANMA! init() kodu yüklenir yüklenmez anında tetikle
  5. Mobil WebGL çökmesini önlemek için düşük polygon sayısı

KAMERA:
  - İzometrik (çapraz) açı
  - Mouse ile sağa-sola kaydırma (Pan) ve zoom
  - Döndürme (Rotate) KESİNLİKLE KAPALI
  - OrbitControls: enableRotate = false

ETKİLEŞİM:
  - Binaya tıklayınca: GSAP ile hafif zıplama animasyonu
  - Tıklanan binanın altında sarı seçim halkası
  - Alttan/yandan detay paneli (bina adı + "Yükselt" butonu)

HER NESNE = AYRI KLASÖR + AYRI KOD PRENSİBİ:
  Her 3D nesne (bina, ağaç, sur, dekorasyon) kendi klasöründe:
    objects/<nesne_adi>/
      ├── model.js      → Three.js geometri + mesh oluşturma
      ├── config.json    → Boyut, renk, seviye, maliyet parametreleri
      └── interact.js    → Tıklama, seçim, yükseltme mantığı

    objects/wall/          → Surlar (arka/yan taraf, şehri korur)
    objects/tower/         → Kuleler (surların köşelerinde)
    objects/tree/          → Akdeniz ağaçları (servi/çam, ince uzun)
    objects/ground/        → Zemin (yeşil çimen düzlemi)
    objects/road/          → Roma yolları (taş döşeli)
    objects/market/        → Pazar binası (ticaret)
    objects/temple/        → Tapınak (inanç sistemi)
    objects/harbor/        → Liman (deniz ticareti)
    objects/ship/          → Gemi (ticaret/savaş)
    objects/barracks/      → Kışla (ordu)
    objects/soldier/       → Asker figürü
    objects/arena/         → Arena (festival)

ARAYÜZ (UI - Tailwind CSS):
  1. Üst bar: Kırmızı + altın sarısı, kaynaklar (Odun, Buğday, Nüfus, Altın)
  2. Bina tepesinde: Kamerayı takip eden mavi kalkan (HTML/CSS), seviye no
  3. Detay paneli: Bina adı, seviye, "Yükselt" butonu (alttan açılır)

AGENT 1-5 SCRİPT YAPISI ($dalgaAdimFaz):
  $dalgaAdimFaz = @{
      X = @{ adim=Y; fazlar=@("AX-Z"); bekle=@(agent_numaralari) }
  }
  adim: Bu agent bu dalgada kaçıncı adımda
  fazlar: Çalıştırılacak faz kodları
  bekle: Önceki adımdaki agent numaraları

AGENT 6 SCRİPT YAPISI (FARKLI!):
  $dalgaAdimHaritasi: Her dalga için adım-agent haritası
  $dalgaFazlari: Her dalga için Agent 6'nın fazları
