================================================================================
🏛️ ROMA CITY BUILDER - MASTER FAZ PLANI
================================================================================
PROJE: Roma İmparatorluğu Şehir Kurma (3D City Builder)
TEKNOLOJİ: Three.js + Tailwind CSS + GSAP + Vanilla JS
FRAMEWORK: YASAK (React, Vue, Angular vb. kullanılamaz)
SUNUCU: YOK (tamamen client-side, tek HTML sayfası)

6 AGENT SİSTEMİ:
  Agent 1 - SCENE_MASTER:    Three.js sahne, kamera, renderer, ışık
  Agent 2 - OBJECT_ENGINE:   3D nesne modelleri (her nesne ayrı klasör)
  Agent 3 - UI_DESIGNER:     HTML/Tailwind arayüz tasarımı
  Agent 4 - INTERACTION:     Tıklama, seçim, animasyon, kamera kontrol
  Agent 5 - RESEARCHER_QA:   Araştırma, test, dokümantasyon
  Agent 6 - ORCHESTRATOR:    Denetim, dalga geçiş onayı (KOD YAZMAZ)

KRİTİK PRENSIP: HER NESNE = AYRI KLASÖR
  objects/<nesne_adi>/model.js     → 3D geometri oluşturma
  objects/<nesne_adi>/config.json  → Parametreler (boyut, renk, seviye, maliyet)
  objects/<nesne_adi>/interact.js  → Tıklama/seçim/yükseltme mantığı

DALGA 1: SAHNE İSKELETİ + TEMEL NESNELER (Temel Yapı)
════════════════════════════════════════════════════════════════════════════════
HEDEF: Three.js sahne çalışsın, zemin görünsün, temel nesneler yerleşsin.

  ADIM 1 (PARALEL):
  Agent 1: index.html (Three.js + Tailwind + GSAP CDN), js/scene.js (sahne),
           js/renderer.js (WebGL renderer, gölge kapalı, pixelRatio=1),

           js/ground.js (yeşil çimen düzlemi), js/main.js (init tetikleyici)
  Agent 2: objects/ground/model.js + config.json (yeşil düzlem),
           objects/senate/model.js + config.json (kubbeli senato iskeleti),

           css/ui.css (ek stiller), js/ui/topBar.js
  Agent 5: Roma mimarisi araştırma dokümanı, shared/contracts/objectRegistry.js
           (tüm nesne isimleri ve tipleri), shared/types/gameTypes.js

  ADIM 2 (Agent 1+2+3+5 sonrası):
  Agent 4: js/interaction/cameraControls.js (pan+zoom, rotate kapalı),
           js/interaction/raycaster.js (temel tıklama algılama)
