GSAP Animasyon Parametreleri:

1. Zıplama (Jump):
   - Özellik: Seçilen nesnenin Y ekseninde kısa süreli yukarı hareket edip yerine geri dönmesi.
   - Parametreler:
     - `duration`: 0.3 (saniye)
     - `y`: y_offset değeri eklenecek (örneğin orijinal_y + 10)
     - `ease`: "back.out(1.7)"
   - Örnek:
     ```javascript
     gsap.to(nesne.position, {
       duration: 0.3,
       y: orijinal_y + zıplama_miktarı,
       ease: "back.out(1.7)",
       onComplete: () => {
         // Yere geri dönme animasyonu eklenebilir
       }
     });
     ```

2. Seçim Halkası Vurusu (Selection Pulse):
   - Özellik: Seçilen nesnenin altında veya etrafında bulunan halkanın sürekli büyüyüp küçülmesi.
   - Parametreler:
     - `duration`: 0.8 (saniye)
     - `scale`: 1 ile 1.1 arasında değişecek
     - `repeat`: -1 (sonsuz tekrar)
     - `yoyo`: true (büyüme ve küçülme işlemini yumuşak geçişle yap)
   - Örnek:
     ```javascript
     gsap.to(halka.scale, {
       duration: 0.8,
       x: 1.1,
       y: 1.1,
       z: 1.1,
       repeat: -1,
       yoyo: true
     });
     ```

3. Panel Kayması (Panel Slide):
   - Özellik: Kullanıcı bir nesne seçtiğinde bilgi panelinin ekranın bir kenarından (genellikle alt veya yan) kayarak gelmesi.
   - Parametreler:
     - `duration`: 0.4 (saniye)
     - `y` veya `translateY`: Kayma miktarı (görünür konuma)
     - `ease`: "power2.out" (hızlı başlayıp yavaşlayarak durma)
   - Örnek:
     ```javascript
     gsap.to(panel, {
       duration: 0.4,
       y: 0, // Hedef konum (örneğin ekranda tam görünür hali)
       ease: "power2.out"
     });
     ```
